# archive/ — полный архив (сбоку)

Здесь лежат **полные копии** источников, когда одной ссылки мало.

Это **не** основная документация. Агенты и люди по умолчанию работают с `docs/` и корневым `README.md`.

## Два слоя репозитория

| Слой | Путь | Роль |
|------|------|------|
| **Основное** | `docs/`, `prompting/`, `roles/`, … | Выжимки, шаблоны, правила — AI-first, actionable |
| **Архив** | `archive/` | Сырьё: снимки статей, клоны репо, сырые материалы |

```
источник (URL / внешний репо)
        │
        ├─► SOURCES.md          # реестр ссылок
        ├─► docs/*.md           # главная выжимка (всегда)
        └─► archive/...         # полная копия (опционально)
```

## Структура

```
archive/
├── README.md                 # этот файл
├── articles/                 # снимки статей (markdown/html)
└── repos/                    # клоны / sparse-copy внешних репозиториев
```

Папки создаются при первом реальном копировании. Пока пусто — всё держим по ссылкам в [`SOURCES.md`](../SOURCES.md).

## For agents

- Сначала читай `docs/` и `SOURCES.md`
- В `archive/` заходи только если в выжимке явно указан путь к архиву или не хватает деталей из docs
- Не дублируй архив в docs; не раздувай контекст сырыми копиями без нужды
