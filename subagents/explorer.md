# Subagent brief: explorer

> Паттерн рядом с built-in **Explore** ([docs](https://cursor.com/docs/subagents)). Кастом нужен только для жёсткого формата отчёта команды.

## For agents (parent)

**Когда спавнить:** широкий поиск «где X»; карта модуля до плана; шумный промежуточный output не нужен в main chat.

**Сначала:** дай шанс built-in Explore. Этот brief — если нужен фиксированный шаблон ответа или project-specific корни поиска.

**В prompt subagent обязательно:**
- Вопрос / цель исследования
- Корни путей (пакеты, apps/)
- Что вернуть (пути + символы + 1–2 фразы «зачем»)
- Лимит: не читать `archive/` / секреты / `node_modules`

---

## Установка

Создай `.cursor/agents/explorer.md`:

| Frontmatter | Значение |
|-------------|----------|
| `name` | `explorer` |
| `description` | Исследует кодовую базу и возвращает сжатую карту путей/символов. Когда нужна разведка без раздувания основного чата. |
| `model` | `inherit` |
| `readonly` | `true` |

**Тело:** исследовать репо, не редактировать. Вернуть только: релевантные пути, ключевые символы/entrypoints, риски/неизвестности, next steps для parent (без реализации).
